Hi all, 到了第四天終於可以來搞專案了，今天的目標很簡單，我想要寫出取得 Block 的 API。

首先來簡單介紹下 Block 的結構:

接下來說明下區塊雜湊值的計算:

Hash = SHA256 ( Timestamp + Previous Hash + Data + Nonce)

其中 Nonce 是個變數值，他會隨著當前的計算困難度( Difficulty) 而有所變化，由於是自己的區塊鏈環境，我們不需要太過強的困難度，因此困難度可視為0。
